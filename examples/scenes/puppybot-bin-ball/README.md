# PuppyBot Bin And Ball

Example scene that uses the reusable PuppyBot robot asset with a floor, trash bin, ball, cameras, and the four-arm-servo ST3215 virtual bus mapping.

Run from the RobotDreams crate root:

```bash
cargo run -- examples/scenes/puppybot-bin-ball/project.json
```

The crate-root `project.json` points at the same scene for convenience.
